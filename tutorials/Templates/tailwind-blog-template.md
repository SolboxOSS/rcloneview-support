# RcloneView 클라우드 동기화 완전 가이드

  이 가이드는 다양한 클라우드 서비스를 손쉽게 동기화하는 방법을 알려드립니다.


## 준비물

- RcloneView 설치본
- 클라우드 계정 (예: Google Drive, Dropbox)
- 기본적인 파일 백업 개념

## 설치 및 설정

```bash
npm install -g rcloneview
rcloneview config
```
