# File management

## 리모트 스토리지 브라우징 하기


- 익스폴로러 창 상단의 + 버튼을 누른 후 브라우징하고자 하는 리모트 스토리지를 선택합니다.
- 처음 리모트를 추가할 경우 오른쪽 익스플로러 창에서 리모트 스토리지 버튼의 Open을 클릭합니다.
- 오픈 후 리모트 스토리지를 브라우징 할 수 있으며, 원격 리모트 스토리지의 파일을 관리할 수 있습니다.

</video>

:::
