# 이미지 크기 조정하기


## Html을 이용한 절대경로 이미지 붙여 넣기

- Small size (width = 200 pixel, high= auto, arrange=left)

```markdown
<img 
  src="/support/images/Pasted%20image%2020250509180458.png" 
  alt="small image" 
  class="img-small img-left" 
/>
```

  class="img-small img-left"
